class nc.NcCriterion(clayers: Sequence[nc.NcLayer], analyzer: nc.NcAnalyzer, **kwds)¶

Bases: engine.LayerLocalCriterion, engine.Criterion4RootedSearch

Neuron coverage criterion

A criterion operates based on an analyzer to find new concrete inputs.

Flag prefer_rooted_search can be used in case both the criterion and the analyzer support the two kinds of search; the default behavior is to select rooted search.

find_next_rooted_test_target() → Tuple[NewType.<locals>.new_type, nc.NcTarget]¶

Seeks a new test target associated with an existing test input taken from the set of recorded test cases.

Note this method MUST perform enough bookkeeping so that two successive calls that are not interleaved with any call to Criterion.add_new_test_cases() return different results. This property is to enforce progress upon unsuccessful search of concrete inputs.
